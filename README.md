# openvpn_proton
OpenVPN / ProtonVPN

## Usage

```
$ git clone https://github.com/vavkamil/openvpn_proton.git
$ cd openvpn_proton
$ pip install -r requirements.txt
$ chmod +x run.sh
$ ./run.sh
```
